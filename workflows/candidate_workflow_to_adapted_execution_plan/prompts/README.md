# `candidate_workflow_to_adapted_execution_plan` Prompts

- `frame_producer.md`: frames one selected workflow plus task context as an adaptation-planning problem.
- `frame_verifier.md`: checks that the adaptation request and acceptance surface are explicit enough for fit analysis.
- `analyze_producer.md`: assesses workflow fit, step-level adaptation notes, and parameterization pressure.
- `analyze_verifier.md`: checks that the fit assessment and step-adaptation matrix are explicit, bounded, and packaging-ready.
- `package_producer.md`: packages the adapted execution plan, proposed workflow parameters, machine-readable summary, and concrete next action.
- `package_verifier.md`: confirms the adapted-execution package is aligned, publication-ready, and still stops at plan publication rather than downstream execution.
