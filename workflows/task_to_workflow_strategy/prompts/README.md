# Task To Workflow Strategy Prompts

- `frame_producer.md` / `frame_verifier.md`: define the task boundary, sponsor, desired outcome, and explicit selection criteria before any route is chosen.
- `select_producer.md` / `select_verifier.md`: compare at least three workflow candidates, include the workflow-builder baseline, and choose one explicit strategy route without executing it.
- `package_producer.md` / `package_verifier.md`: turn the selected route into a terminal strategy package, machine-readable summary, and next-action artifact.
