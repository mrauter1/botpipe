# Security Remediation Prompts

- `assessment_producer.md` / `assessment_verifier.md`: assess the adopted evidence pack as a security finding and produce exploit, root-cause, and remediation-option artifacts.
- `remediation_producer.md` / `remediation_verifier.md`: choose and plan the verified remediation, including verification, rollout, and rollback-safety artifacts.
- `closure_producer.md` / `closure_verifier.md`: package the final remediation narrative, stakeholder communication draft, and closure-evidence contract.
