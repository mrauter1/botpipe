# Evidence Pack Checklist

- Scope brief defines the trigger, downstream consumer, in-scope boundary, out-of-scope boundary, and source constraints.
- Investigation objectives define what downstream work this evidence pack must enable.
- Evidence intake register lists expected sources, hinted sources, and known missing proof.
- Source inventory records each inspected source and what it contributed.
- Coverage matrix maps objectives or questions to supporting evidence and unresolved gaps.
- Findings distinguish confirmed facts from weaker indications.
- Gap register explains why each unresolved gap matters and what proof is still missing.
- Evidence pack summarizes scope, sources reviewed, key findings, unresolved gaps, and downstream handoff notes.
- `evidence_pack_summary.json` is valid JSON and matches the durable evidence artifacts.
