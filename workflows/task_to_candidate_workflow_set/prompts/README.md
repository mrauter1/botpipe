# `task_to_candidate_workflow_set` Prompts

- `frame_producer.md`: frames the task as a candidate-workflow selection problem.
- `frame_verifier.md`: checks that the framing package is explicit enough for capability-backed comparison.
- `analyze_producer.md`: compares current workflow candidates, ranks them, and explains the portfolio posture.
- `analyze_verifier.md`: checks that the ranked comparison and posture are explicit, evidence-backed, and route-safe.
- `package_producer.md`: packages the ranked candidate set into durable human-facing and machine-readable artifacts.
- `package_verifier.md`: confirms the candidate-workflow-set package is ready for deterministic publication.
